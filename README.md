## Setup

Install dependencies

```
nvm install
yarn install
brew bundle
yarn generate:ssl
```

## Run

All project processes can be started using the following:

```
yarn start
```

## Testing

Runs all the automated QA tools

This can be run using:

```
yarn test
```

## Build

Generate a production build of the site using the following:

```
yarn build
```
